리눅스 문제
bash쉘 기준

Echo 명령어
문제 hello bash 를 출력하라
답 echo hello bash

Pwd 명령어
문제 pwd 를 이용하여 현재 경로를 출력하라
답 pwd

Cd 명령어
문제 : cd 명령어를 이용하여 디렉토리를 이동하라 (상대경로 ,절대경로)
예시 : 
	cd .
	cd ..
	cd /usr/bin
	cd /usr
	cd bin

Ls 명령어
문제 : ls 명령어를 이용하여 디렉토리 정보를 출력하라
예시 :
	ls (현재 디렉토리)
	ls <상대경로or 절대경로>
	ls -a (숨김파일 포함), ls -l (권한 포함) , ls -al

Touch 명령어
문제 : touch 파일에 대해 수정 및 생성을 할수있다.
예시 : touch <파일명>

File 명령어
문제 : file 명령어를 이용하여 file에 대한 정보를 얻을 수 있다.
예시 : file <파일명>

Cat 명령어
문제 : cat 명령어를 이용하여 파일 내용물을 보여준다.
예시 : 
	cat <파일명>
	cat <파일명1> <파일명2> 두개 의 파일내용을 연달아서 보여준다.

History 명령어
문제 : history 명령어를 이용하여 실행했던 명령어들을 검색하라
예시 : history
