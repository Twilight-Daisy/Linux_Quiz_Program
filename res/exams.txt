1.숨겨진 파일이나 디렉토리를 보여주는 명령어를 입력하시오
ls-a

2.권한이나 파일 크기,수정일자 파일이름 등을 출력하는 명령어를 입력하시오
ls-l

3.디렉토리를 알파벳 역순으로 출력하는 명령어를 입력하시오
ls-r

4.현재 디렉토리에 있는 모든 파일 및 디렉토리의 자세한 정보를 출력하는 명령어를 입력하시오
ls-al

5.현재 작업중인 디렉토리를 보여주는 명령어를 입력하시오
pwd

6.디렉토리를 이동시켜주는 명령어를 입력하시오
cd

7.상위 디렉토리로 이동하는 명령어를 입력하시오
cd ..

8.디렉토리를 생성하는 명령어를 입력하시오
mkdir

9.비어있는 디렉토리를 삭제하는 명령어를 입력하시오
rmdir

10.파일이나 디렉토리를 복사하는 명령어를 입력하시오
cp

11.디렉토리 전체를 복사하는 명령어를 입력하시오
cp -r

12.파일을 이동시키는 명령어를 입력하시오
mv

13.파일이나 디렉토리를 삭제하는 명령어를 입력하시오
rm

14.파일의 종류를 출력하는 명령어를 입력하시오
file

15.파일에 내용을 입력하거나 내용을 보여주는 명령어를 입력하시오
cat

16.새로운 빈 파일을 생성하거나 파일의 타임스탬프를 변경하는 명령어를 입력하시오
touch

17.여태 실행했던 명령어들을 검색하는 명령어를 입력하시오
history

18.디렉토리 또는 파일의 크기를 출력하는 명령어를 입력하시오
du

19.두 파일을 비교하여 처음으로 다른 문자가 나타나는 위치를 화면에 출력하는 명령어를 입력하시오
cmp

20.여러 개의 파일을 하나의 파일로 결합하는 명령어를 입력하시오
tar

21.터미널 화면을 깨끗하게 비워주는 명령어를 입력하시오
clear

22.파일을 나누는 명령어를 입력하시오
split

23.파일의 처음 부분을 출력하는 명령어를 입력하시오
head

24.파일의 끝 부분을 출력하는 명령어를 입력하시오
tail

25.파일의 접근 권한을 바꾸는 명령어를 입력하시오
chmod

26.문자열을 찾는 명령어를 입력하시오
grep

27.현재 접속 사용자를 보여주는 명령어를 입력하시오
users
